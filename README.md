# Practica de Entornos: Proyecto de Ivan y Konstantin
Autores: Ivan Malyshev, Konstantin Petrenko

# Cómo usar?
Paso por paso.

- Paso 1: Pulsa "Code" y "Download ZIP" (o similar en espanol si es tu lenguaje por defecto)
- Paso 2: Encuentra tu archivo cargado y desempaquetalo en un sitio preferido.
- Paso 3: Debes tener un IDE instalado. Recomendamos usar Intellij IDEA.
- Paso 4: Abre tu IDE, naviga a "File" -> "Open" y busca tu carpeta con proyecto.
- Paso 5: Aqui lo tienes!
